# Le Wololo Challenge

React app for managing teams and players.

**Live site:** https://alexdissaux.github.io/ordreduwololo/

## Deploy to GitHub Pages

```bash
# 1. Test locally
npm run dev

# 2. Commit changes
git add .
git commit -m "your message"
git push

# 3. Deploy
npm run deploy
```

Site updates in 1-5 minutes.
